# Descripción del ejercicio.

Se creo un repositorio con el fin de hacer práctico el uso del git y github, incluyendo comandos esenciales, trabajos de ramificación, opciones de colaborador, integración con PR y resolución de conflictos, logrando así demostrar el dominio que se tiene con el uso de git y github.
